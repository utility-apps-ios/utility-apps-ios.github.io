Help & Support

Getting Started

AMC Math Practice helps you prepare for the American Mathematics Competition (AMC 8, 10, and 12) with real past problems and detailed solutions.

The app offers two main modes:

  • Test Mode - Simulate real test conditions with a timer
  • Review Mode - Practice at your own pace with immediate feedback

Using Test Mode

Test Mode simulates actual AMC testing conditions to help you practice time management.

Starting a Test

  • Select your desired test (AMC 8, 10, or 12) and year
  • Set a custom duration using the time picker (or use the default)
  • Tap "Start Test" to begin

During the Test

  • The timer displays remaining time at the top
  • Timer turns red when 5 minutes or less remain
  • Use Pause/Resume to take breaks if needed
  • Answer by tapping A, B, C, D, or E
  • Tap "End Test" when you're finished

After the Test

View detailed results showing your score, time spent on each problem, and access to solutions.

Using Review Mode

Review Mode is perfect for focused practice without time pressure.

  • No time limit - take as long as you need
  • Get immediate feedback after each answer
  • Access the solution for every problem
  • Great for learning new problem-solving techniques

Navigation Tips

Moving Between Problems

  • Use Next and Previous buttons to move one problem at a time
  • Tap the Grid icon to see all problems and jump to any one

Problem Status Colors

  • Gray - Unanswered
  • Blue - Answered
  • Orange border - Flagged for review

Flagging Problems

Tap the flag icon to mark problems you want to revisit. Flagged problems show an orange border in the grid view.

iPad Keyboard Shortcuts
Use arrow keys (← →) to navigate between problems. Press A, B, C, D, or E to quickly select an answer.

Understanding Results

Score Summary

After completing a test, you'll see:

  • Total score and percentage
  • Number of correct, wrong, and blank answers
  • Total time spent

Problem-by-Problem Breakdown

The results table shows each problem with:

  • Your answer vs. the correct answer
  • Time spent on each problem
  • Tap the eye icon to view the problem and its solution

Time Color Coding

  • Green - Quick (good pace)
  • Yellow - Moderate time
  • Red - Took longer than average

Tap any column header to sort the results by that column.

Test History

Access your past tests from the History tab on the main screen.

  • View a summary of all completed tests
  • Tap any test to see full details and review problems
  • Swipe left on a test to delete it
  • Use the trash icon to clear all history

Frequently Asked Questions

How is my score calculated?

The app uses official AMC scoring rules:

  • AMC 8: 1 point per correct answer, 0 for blank or wrong. Max score: 25. No penalty for guessing.
  • AMC 10/12: 6 points per correct answer, 1.5 points for blank, 0 for wrong. Max score: 150. Since leaving a question blank earns 1.5 points, there's a strategic penalty for guessing incorrectly.

Can I change the test duration?

Yes! Before starting a test, you can set any custom duration using the time picker. This lets you practice with shorter or longer time limits as needed.

Where do the problems and solutions come from?

Problems are from official past AMC competitions, copyrighted by the Mathematical Association of America (MAA). Solutions are sourced from the Art of Problem Solving (AoPS) Wiki, available under Creative Commons BY-SA license.

Is my data stored online?

No. All your test history and results are stored locally on your device only. Nothing is uploaded to any server, and no account is required.

Can I use this app offline?

The problems work offline, but viewing solutions requires an internet connection to load from the AoPS Wiki.
